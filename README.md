**Media Unit Of Negombo Southernians** - Creative media unit capturing stories frame by frame.

## 🎬 About

MUONS is the official media unit of Negombo South International School, dedicated to documenting school events, sports meets, performances, and creative showcases through professional photography and videography.

## 🚀 Features

- **Responsive Design** - Fully optimized for mobile, tablet, and desktop
- **Modern UI/UX** - Cyberpunk-inspired aesthetic with glass morphism effects
- **Smooth Animations** - Canvas-based aurora and network animations
- **Accessibility** - WCAG compliant with keyboard navigation support
- **Performance** - Optimized loading and rendering
- **SEO Friendly** - Semantic HTML and meta tags

- **HTML5** - Semantic markup
