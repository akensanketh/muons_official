Welcome to the central hub for **MUONS**. This portal is a high-tech corporate interface designed for the Media Unit of Negombo Southernians, managed by **Aken Sanketh**.

## 🧬 Project DNA
* **Theme:** Carbon background with Cyan accents.
* **Vibe:** Cyberpunk meets Corporate.
* **Typography:** Orbitron (Headings) & JetBrains Mono (Technical/UI).
* **Core Principle:** #PROVE. #THRIVE. #OVERCOME.

## 🚀 Deployment Status
The site is live and managed via GitHub Pages.
**Live URL:** [https://akensanketh.github.io/muons_official/](https://akensanketh.github.io/muons_official/)

## 🛠️ Usage
This repository contains the source code for the MUONS official portal.
- **index.html**: Main landing interface.
- **css/muons-override.css**: The Carbon & Cyan stylistic core.
- **assets/**: Media assets including photography and videography links.
